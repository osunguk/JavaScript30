### border-radius

### box-shadow



*작동 순서*

1. setInterval 함수로 1초에 한번씩 setDate 함수 실행
2. setDate에서 new Date()에서 시간 데이터 확득
   getSecond(), getMinutes(), getHours()
3. 시간 데이터를 토대로 각 초침, 분침, 시침, 각도 계산
4. transform: rotate(deg)에 계산한 값을 넣어 시계처럼 작동



*추가할 만한 사항*

- 침 길이 바꾸기
- 초침이 90도에 가면 생기는 오류 잡기
- 시계 디자인 바꾸기
